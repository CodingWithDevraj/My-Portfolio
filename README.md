## How to Run Locally

1. Clone the repository

```

git clone https://github.com/CodingWithDevraj/My-Portfolio

```

2. Navigate to project folder

```

cd your-portfolio-repo

```

3. Install dependencies

```

npm install

```

4. Run the app

```

npm start

```
